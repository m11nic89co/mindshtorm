# canvases/

Папка для **референсных** JSON Canvas-схем (Obsidian-формат, без обёртки `.mindstorm`).

| Источник | Назначение |
|----------|------------|
| `src/lib/demoCanvas.ts` | Встроенные демо: `getDemoCanvas('ru')`, `getDemoCanvas('en')` |

Сюда можно класть свои `.canvas` для разработки и тестов загрузки.
