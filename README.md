# MindShtorm

Интерактивная доска для брейншторма в стиле Obsidian Canvas. Формат **JSON Canvas** — доски открываются в Obsidian.

**Онлайн:** https://m11nic89co.github.io/mindshtorm/

## Возможности

- Карточки с markdown-текстом
- Связи между идеями
- Группы и цветовые метки (как в Obsidian)
- Импорт и экспорт `.canvas`
- Автосохранение в браузере

## Локальный запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:5173

## Сборка

```bash
npm run build
npm run preview
```
